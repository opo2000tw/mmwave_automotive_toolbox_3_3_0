Version 1.0.0 ============= Overview ======== This lab demonstrates the use of TI mmWave sensors to detect children and adults inside a vehicle cabin using the xWR6443ISK or xWR6843ISK sensor modules in the roof mounted position. The lab provides a CCS project as well as a visualizer to process the data. Features ======== The following features are supported: * Supports xWR6843ISK antenna module. * Supports xWR6443 device. * GUI displays point cloud and occupancy information. * Supports mmWave SDK 3.5.0.4.
