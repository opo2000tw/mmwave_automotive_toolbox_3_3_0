Version 1.0.1 ============= **NOTE**: Due to the ongoing social distancing guidelines and TI policies, the scope of testing for this release was limited to 1-2 person tests along with the Ashton Drake breathing doll. Overview ======== This lab demonstrates the use of TI mmWave sensors to detect children, adults and animals inside a vehicle cabin using the AWR6843AOP or AWR6843ISK-ODS sensor modules in the roof mounted position. The lab provides full source code and CCS projects for building the binaries from source. Features ======== The following features are supported: * Supports AWR6843AOP antenna module. * Supports AWR6843ISK-ODS antenna module. * GUI displays point cloud and occupancy information. * Supports mmWave SDK 3.5.0.4. New and Updated Features ======================== * Updated to mmWave SDK 3.5.0.4. Resolved Incident Reports ========================= The following are Incident Reports resolved in this release: * V1.0.1: Resolved broken links in documentation, missing CLI command in ODS cfg, and replay capability for GUI.
